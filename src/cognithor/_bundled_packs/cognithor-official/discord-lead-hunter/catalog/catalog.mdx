## Discord Lead Hunter

A free, bundled pack that scans Discord channels and scores messages with your local LLM.
Requires a Discord bot token set as the `COGNITHOR_DISCORD_TOKEN` environment variable.
If no token is configured, `scan()` returns an empty list gracefully.
