**Voraussetzungen:** Python 3.12+, ~2 GB freier Plattenplatz, Internet-Verbindung für den initialen `pip install` und den Ollama-Modell-Download.

**Endzustand nach Schritt 01:** Eine lauffähige PKV-Analyse-Crew aus zwei Agenten, die lokal mit Ollama läuft und einen strukturierten Markdown-Report produziert.

**Lauffähige Beispiele:** Alle Code-Snippets dieser Quickstart liegen als eigenständige Python-Projekte unter [`examples/quickstart/`](../../examples/quickstart/) im Repo — inklusive `test_example.py`, die jeder CI-Lauf automatisch smoke-testet.
