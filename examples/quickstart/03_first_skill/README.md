# 03 · First Skill — runnable example

Bundle a tool, a persona, and trigger keywords into a reusable skill via
`@agent`. Walks through
[`docs/quickstart/03-first-skill.md`](../../../docs/quickstart/03-first-skill.md).

## Run it

```bash
pip install -r requirements.txt
python main.py
```

## Smoke-test it

```bash
pip install pytest pytest-asyncio
python -m pytest test_example.py -v
```
